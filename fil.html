Сұрақ: SQL-де индексті жою үшін:

: DROP INDEX

Сұрақ: SQL-де деректер типтерін анықтау не үшін қажет?

: сақтау пішімін және жады көлемін анықтау үшін

Сұрақ: SQL-де мәтіндік деректер типі:

: VARCHAR

Сұрақ: SQL-де бүтін сандарды сақтау үшін:

: INTEGER

Сұрақ: SQL-де ондық бөлшек сандарды сақтау үшін:

: DECIMAL

Сұрақ: SQL-де күн мен уақытты сақтау үшін:

: DATETIME

Сұрақ: SQL-де жаңа деректер қорын құру үшін:

: CREATE DATABASE

Сұрақ: SQL-де деректер қорын жою үшін:

: DROP DATABASE

Сұрақ: SQL-де VIEW не үшін қолданылады?

: виртуалды кесте жасау үшін

Сұрақ: SQL-де VIEW құру операторы:

: CREATE VIEW

Сұрақ: SQL-де VIEW-ді жою үшін:

: DROP VIEW

Сұрақ: SQL-де триггер не үшін қолданылады?

: автоматты түрде әрекет орындау үшін

Сұрақ: SQL-де триггер жойылатын оператор:

: DROP TRIGGER

Сұрақ: SQL-де кестенің алғашқы кілті не үшін қажет?

: жазбаны бірегей сәйкестендіру үшін

Сұрақ: SQL-де сыртқы кілт (FOREIGN KEY) не үшін қолданылады?

: кестелер арасындағы байланысты орнату үшін

Сұрақ: SQL-де шектеу қою үшін қандай сөз қолданылады?

: CONSTRAINT

Сұрақ: SQL-де деректердің тұтастығын сақтау үшін қолданылады:

: PRIMARY KEY және FOREIGN KEY

Сұрақ: SQL-де деректердің қайталанбауын қамтамасыз ететін кілт:

: PRIMARY KEY

Сұрақ: SQL-де бағанға мән енгізбеуге тыйым салатын шектеу:

: NOT NULL

Сұрақ: SQL-де өріске бастапқы мән орнату үшін:

: DEFAULT

Сұрақ: SQL-де мәннің белгілі бір диапазонда екенін тексеретін шектеу:

: CHECK

Сұрақ: SQL-де деректер қорына кіру құқығын беру үшін:

: GRANT

Сұрақ: SQL-де қол жеткізуді алып тастау үшін:

: REVOKE

Сұрақ: SQL-де пайдаланушылардың құқықтарын басқару не үшін қажет?

: қауіпсіздікті қамтамасыз ету үшін
